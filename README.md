## replicating
just copy them to you .config and /usr/local/bin and also in /etc for some "the finals" fixes
``
cp -r sway/ cmus/ foot/ nwg-look/ qt6ct/ rofi/ gtk-3.0/ MangoHud/ nvim user-dirs.dirs ~/.config

cp -r etc/* /etc/
``
